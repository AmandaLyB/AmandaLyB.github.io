June

First Update! The goal of this project is to create a fully functioning 3D game in the open world survival craft genre with support for multiplayer as a learning experience and a starting point for more complex projects. The player will be able to harvest materials, craft items, fight enemies, and explore the world by completing challenges and puzzles. The project will be considered complete when it is Steam-ready.

So far, I have created the 3D Unreal Engine 5.2 project using the Blueprint Editor and installed the Craft Resource Icons Marketplace content to get started with creating an inventory system. The character model and animations are from SmartPoly's UE Course, which will be the basic guide for the project. The character's idle, walk, and run animations have been hooked up for both first and third person.

The basic inventory system is currently in progress, with drag and drop functionality set up for the player's inventory. The hotbar is still in development.
