Why Godot?

  • Open source for personal or commercial use
  • User-friendly, simple interface
  • Provides well-structured and detailed learning materials
  • Allows coding with multiple languages
  • Features include networking, multiple rendering methods, and an extensive library of plugins
  • Ships to Windows, macOS, Linux, Android, iOS, browsers, and consoles
  • Supports 3D and 2D environments

Godot is a viable alternative to other game engines for simple, beginner-friendly games in either 2D or 3D. While Unreal Engine excels at creating AAA style 3D games, it is also extremely complicated and better suited to graphically-intense games. Unity is perfect for creating 2D games, but is also more complex than Godot. The goal of this project is to learn the basics of game development by utilizing the excellent tutorials provided by the Godot developers. The techniques learned from Godot will apply to other game engines as well, and provide valuable insight into the game development industry.

The Game

The idea behind this game is to incorporate my favorite gameplay and UI elements of games that I have played into a well-rounded and fun experience. The gameplay loop will revolve around building, surviving, and taming various animals as found in Valheim, Arc Survival Evolved, VRising in a multiplayer experience with some creative elements found in games like Minecraft.

  • Intuitive UI
    • Menus for building will be concise and fluid
    • Pre-defined inventory tabs to separate items logically
    • Clothes and armor
  • Invetory
    • Less constrictive weight and space limits for inventory as player progresses (building should be fun, not a chore)
  • Combat
    • Skill-based
    • Variety of player skills and combinations
    • Variety of enemy skills and combinations
  • Taming, breeding, and controlling animals
    • Should be fun and rewarding at all stages of gameplay
    • Taming food will be intuitive based on animal's natural game environment
    • Breeding will create animals with equal or greater stats
    • Controlling animals must be simplistic (no giant, multi-level selection wheels)
