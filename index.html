Progress

May

Created 2D project in Unity and imported the Sprout Lands free game assets. The player character has a player movement script attached with up-down-left-right controls using the WASD keys and the appropriate animations are attached for both walking and idle states. Additionally, a tilemap has been added with a dirt, grass, and path layer using the Sprout Lands tilemap sheets.
